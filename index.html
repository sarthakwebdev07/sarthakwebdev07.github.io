What is JavaScript?

JavaScript is the programming language of the web. Every interactive element you see on a website — buttons that respond to clicks, forms that validate input, menus that slide open — JavaScript powers all of it.

Originally created in 1995, JavaScript has grown into one of the most popular programming languages in the world. Today it runs not just in browsers, but on servers (Node.js), mobile apps, and even desktop software.

💡 Why learn JavaScript first? JavaScript is the only language that runs natively in every browser. If you want to build websites or web apps, JavaScript is non-negotiable — making it the best starting point for most beginners.

Variables & Data Types

A variable is a container that stores data. In modern JavaScript, you declare variables using let, const, or var.

variables.js
// const — value cannot be reassigned
const name = "Sarthak";
const age = 22;
const isDeveloper = true;

// let — value can change
let score = 0;
score = 100; // ✅ allowed

// JavaScript Data Types
const text    = "Hello";      // String
const number  = 42;          // Number
const flag    = true;        // Boolean
const nothing = null;        // Null
const arr     = [1, 2, 3];   // Array
const obj     = { x: 10 };   // Object

Functions

Functions are reusable blocks of code. You define a function once and call it as many times as you need. JavaScript offers several ways to write functions:

functions.js
// 1. Regular function
function add(a, b) {
  return a + b;
}

// 2. Arrow function (modern syntax)
const multiply = (a, b) => a * b;

// 3. Function with default parameter
const greet = (name = "stranger") => {
  return `Hello, ${name}!`;
};

console.log(add(5, 3));       // 8
console.log(multiply(4, 6));  // 24
console.log(greet());         // Hello, stranger!
console.log(greet("Sarthak")); // Hello, Sarthak!

DOM Manipulation

The DOM (Document Object Model) is how JavaScript talks to your HTML. You can find elements, change their content, add styles, and respond to user events — all in real time.

dom.js
// Select elements
const btn   = document.querySelector("#myButton");
const box   = document.querySelector(".output");

// Change content and style
box.textContent = "Updated text!";
box.style.color = "#58A6FF";

// Listen for a click
btn.addEventListener("click", () => {
  box.textContent = "Button was clicked! 🎉";
});

// Create a new element
const para = document.createElement("p");
para.textContent = "I was created with JavaScript.";
document.body.appendChild(para);

Async Programming & Promises

JavaScript is asynchronous — it can start a task (like fetching data from an API) and continue running other code while waiting for a response. This is done using Promises and async/await.

async.js
// Fetch data from an API using async/await
async function getUser(id) {
  try {
    const response = await fetch(`https://api.example.com/users/${id}`);
    const data = await response.json();
    console.log(data.name);
  } catch (error) {
    console.error("Error:", error);
  }
}

getUser(1); // Runs without blocking other code
⚠️ Common Beginner Mistake Don't use var — it has confusing scoping rules. Always prefer const by default, and use let only when you need to reassign the variable.

What is C++?

C++ is a general-purpose, compiled programming language known for its blazing speed and low-level memory control. Created by Bjarne Stroustrup in the 1980s as an extension of C, it powers game engines (Unreal Engine), operating systems, browsers, and financial systems.

If JavaScript is the language of the web, C++ is the language of performance-critical software.

💡 Where is C++ used? Game development (Unreal Engine, AAA games), system software (OS kernels, drivers), embedded systems, competitive programming, and high-frequency trading systems.

Syntax & Basic Structure

Every C++ program starts with a main() function. Unlike JavaScript, C++ is statically typed — you must declare the type of every variable.

basics.cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
  // Variable declarations (type must be specified)
  string name   = "Sarthak";
  int    age    = 22;
  double gpa    = 3.9;
  bool   passed = true;

  // Output to console
  cout << "Name: " << name << endl;
  cout << "Age: "  << age  << endl;

  // If-else
  if (passed) {
    cout << "Congratulations!" << endl;
  } else {
    cout << "Try again." << endl;
  }

  return 0; // 0 = success
}

Pointers & Memory Management

The most powerful — and most feared — feature of C++ is manual memory management using pointers. A pointer stores the memory address of another variable.

pointers.cpp
#include <iostream>
using namespace std;

int main() {
  int x = 42;
  int* ptr = &x;  // ptr holds the address of x

  cout << "Value of x:   " << x    << endl; // 42
  cout << "Address of x: " << &x   << endl; // e.g. 0x7ff...
  cout << "ptr points to:" << *ptr << endl; // 42 (dereference)

  // Dynamic memory allocation
  int* heap = new int(100);  // allocate on heap
  cout << *heap << endl;      // 100
  delete heap;                // always free memory!

  return 0;
}

Object-Oriented Programming in C++

C++ supports full OOP with classes, inheritance, polymorphism, and encapsulation. Here's a complete class example:

classes.cpp
#include <iostream>
using namespace std;

class Animal {
private:
  string name;
  int age;

public:
  // Constructor
  Animal(string n, int a) : name(n), age(a) {}

  void speak() {
    cout << name << " makes a sound." << endl;
  }
  string getName() { return name; }
};

// Inheritance
class Dog : public Animal {
public:
  Dog(string n) : Animal(n, 0) {}

  void speak() {  // override parent method
    cout << getName() << " says: Woof!" << endl;
  }
};

int main() {
  Dog dog("Buddy");
  dog.speak(); // Buddy says: Woof!
  return 0;
}
⚠️ Memory Leak Warning Every new must have a matching delete. Forgetting to free memory causes memory leaks — a common C++ bug. Use smart pointers (unique_ptr, shared_ptr) in modern C++ to avoid this.

What is Python?

Python is a high-level, beginner-friendly programming language famous for its clean, readable syntax. Created by Guido van Rossum in 1991, Python has become the go-to language for data science, AI, machine learning, web development, and automation.

Its philosophy: code should be readable and beautiful. Python reads almost like plain English, which makes it one of the best first languages to learn.

💡 Where is Python used? AI & Machine Learning (TensorFlow, PyTorch), data analysis (Pandas, NumPy), web backends (Django, FastAPI), scripting & automation, and scientific computing.

Syntax & Basic Concepts

Python uses indentation (spaces/tabs) instead of curly braces to define code blocks. There are no semicolons, and variable types are inferred automatically.

basics.py
# Variables — no type declaration needed
name   = "Sarthak"
age    = 22
height = 5.11
active = True

# Print to console
print(f"Hello, {name}! You are {age} years old.")

# If-elif-else
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teenager")
else:
    print("Child")

# For loop
for i in range(5):
    print(f"Count: {i}")

# While loop
count = 0
while count < 3:
    print("Looping...")
    count += 1

Lists, Dictionaries & Comprehensions

Python's built-in data structures are powerful and easy to use. Lists are ordered collections; dictionaries store key-value pairs.

data_structures.py
# Lists
langs = ["JavaScript", "C++", "Python"]
langs.append("Rust")        # add item
langs.remove("C++")         # remove item
print(langs[0])              # "JavaScript"
print(langs[-1])             # "Rust" (last item)

# List comprehension (Python superpower!)
squares = [x**2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]

# Dictionary
student = {
    "name": "Sarthak",
    "age": 22,
    "skills": ["HTML", "CSS", "JS"]
}
print(student["name"])         # Sarthak
student["grade"] = "A"        # add new key

# Iterate over dict
for key, value in student.items():
    print(f"{key}: {value}")

Object-Oriented Programming in Python

Python's OOP is clean and intuitive. Classes use self to refer to the current instance, and __init__ is the constructor method.

classes.py
class Animal:
    def __init__(self, name, species):
        self.name    = name
        self.species = species

    def speak(self):
        print(f"{self.name} makes a sound.")

    def __repr__(self):
        return f"Animal({self.name}, {self.species})"


# Inheritance
class Dog(Animal):
    def __init__(self, name):
        super().__init__(name, "Canis lupus")

    def speak(self):  # override
        print(f"{self.name} says: Woof! 🐶")


dog = Dog("Buddy")
dog.speak()        # Buddy says: Woof! 🐶
print(dog.species)  # Canis lupus
💡 Python Pro Tip Use f-strings (f"Hello {name}") instead of string concatenation — they're faster, cleaner, and more readable. Available in Python 3.6+.
